**Areas discussed:** Landing density, Public nav + CTA behavior, Auth surface style, Copy tone

---

## Landing density

| Option | Description | Selected |
|--------|-------------|----------|
| Marketing-rich Stitch landing | Keep a fuller branded landing with hero plus supporting sections, but make CTA paths decisive into the product. | ✓ |
| Minimal redirect landing | Compress the page into a fast redirect shell that mostly pushes users straight into auth. | |

**User's choice:** Marketing-rich Stitch landing with clear conversion paths.
**Notes:** The chosen direction keeps the exported Stitch energy instead of reducing the landing page to a thin wrapper. CTA behavior still needs to pull users quickly into auth or dashboard.

---

## Public nav + CTA behavior

| Option | Description | Selected |
|--------|-------------|----------|
| Simple conversion nav | Keep public navigation lightweight and useful; signed-out users go to auth, signed-in users go to dashboard. | ✓ |
| Expanded marketing nav | Use the redesign to introduce broader public navigation branches and exploratory paths. | |

**User's choice:** Simple conversion nav.
**Notes:** The redesign should not create new public-product branches. Existing entry behavior remains the anchor.

---

## Auth surface style

| Option | Description | Selected |
|--------|-------------|----------|
| Focused form card with moderated Stitch atmosphere | Keep the form as the main focus while using collage/background/layer treatments to carry the brand. | ✓ |
| Fully immersive collage-heavy auth | Push visual energy so far that the form shares equal attention with the decorative background. | |

**User's choice:** Focused form card with moderated Stitch atmosphere.
**Notes:** Trust and completion clarity matter more than maximum visual spectacle on login/register.

---

## Copy tone

| Option | Description | Selected |
|--------|-------------|----------|
| Playful landing, calmer auth | Keep Vietnamese-first personality throughout, but let auth be one step calmer and clearer than landing. | ✓ |
| Equally loud everywhere | Push the loudest Gen Z tone equally across landing and auth. | |
| More neutral/public-product tone | Pull both landing and auth toward a more conventional and restrained brand voice. | |

**User's choice:** Playful landing, calmer auth.
**Notes:** Auth should remain warm and youthful, but trust and readability take priority. Broken Vietnamese encoding on the current auth pages should be corrected as part of the redesign.
